
# News MCQ Generator (Streamlit App)

## How to Run (in VS Code or terminal):

1. Make sure Python 3.8+ is installed.
2. Open terminal in this folder.
3. Run: pip install -r requirements.txt
4. Then: streamlit run app.py
5. The app will open in your browser at http://localhost:8501

## Features
- Pulls real-time news using Currents API
- Summarizes the news article
- Generates 1 MCQ based on the summary

Enjoy!
